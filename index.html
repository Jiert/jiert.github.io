Moboom

Moboom is a development platform that offers a variety of products which aim to simplify the process of developing and hosting fast, responsive, and beautiful websites. Built with Backbone.js, products like Homepage and Sitekit allow users to choose a theme, enter some information, and publish with ease. DevKit allows more experienced designers and developers to use Moboom's studio, an immersive drag and drop single page application with the ability to create custom widgets.

Backbone Underscore Handlebars Grunt jQuery Bootstrap LESS Node Require Coffeescript
